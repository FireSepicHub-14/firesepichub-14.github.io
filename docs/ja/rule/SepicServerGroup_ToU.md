# SepicServer Group利用規約・免責事項

SepicServer Group(以下SSGと呼称)とは、以下のサービスが含まれています。

- SepicServer (MainDiscord)
- SepicServer (Minecraft)
- SepicServer Lab (開発Discord)
- SepicServer 運営管理グループ (AdminDiscord)

SepicServer Group利用規約ではSSGに属するすべてのサービスを利用するときに共通する利用規約です。
サービスそれぞれに個別の利用規約も存在します。併せてお読みください。

!!! Warning "注意!!"
    違反した場合は、==Kick== や ==Ban== となります。

<br><br>

## Group全体
SepicServer Groupの利用規約・免責事項です。
```
1.  Discordの規約を守る
    ※(Discordの規約が優先されます)
2. 本Groupで提供する情報には、特定のサービスや商品に関するレビュー、体験談、個人的な見解が含まれますが、その内容の正確性、完全性、有用性を保証するものではありません。
3.  以下の行為は禁止します。
    1.  スパム
    2.  即抜け
    3.  個人情報の拡散
    4.  荒らし行為やその勧誘など(即BAN)
    5.  誹謗中傷、詐欺、酷い嘘、嫌がらせ、差別
    6.  NSFWコンテンツの投稿
    7.  不要不急なメンション
    8.  自己中心的な発言
    9.  無理な要求
    10. 不快になる発言
    11. 不快になるニックネームやアイコンの設定
    12. 法に抵触する行為
    13. このSSG以外から揉め事を持ち込むこと
    14. サブ垢を対処逃れ目的で使用すること(即BAN)
3.  以下の行為は基本的には禁止です。(条件あり)
    1.  引き抜き行為(@souに連絡を)
    2.  宣伝行為(@souに連絡を)
4.  個人情報等をチャットで送信しないでください。揉め事が増えるだけです。
5.  SSG内で添付されたファイルにより問題が生じた場合、運営は責任を負いません。
6.  SSGのメンバー間で起こったトラブル又は、運営と関係のないトラブルは一切の責任を負いません。
7.  SSGのいずれかのサービスでBANの対処となった場合は、全てのサービスでBANとなります。
8.  誤BANを除き、BANの対処となった場合は異議申し立てを受け付けません。
9.  SepicServer Groupのサービスを使用した時点で、SepicServer Group利用規約に同意したものとします。
10.  ルールは予告なしに変更される場合があります。
```
!!! Warning "注意!!"
    禁止行為を行った場合、違反点数が加算されます。
    違反点数が一定以上になった場合、BAN等の対処を行います。
    例外として直ちにBANする場合があります。

!!! Note "補足"
    質問や何か運営に報告したいことがあればメンションして伝えてください

## Discord Server
SepicServer Discord全般の利用規約です。
```
1. 以下の行為は禁止します。
    1. Botのコマンドを連発して打つ
    2. リアクションで反応するBotで遊ぶ(壊れる)
```
<br><br>

## Minecraft Server
SepicServer Minecraftの利用規約・免責事項です。
```
1. 以下の行為は禁止します。
    1. チートクライアント及び、チートModの使用
    2. 他人の建築物の破壊行為
    3. 窃盗行為
2. 推奨環境以外での参加によるトラブルは対応致しません。
3. 推奨環境以外での誤BANは、お許しください。(システムの都合上)
```

!!! Note "補足"
    前提として、弊サーバーの推奨環境はvanilla / Fabricです。
