# 宣伝行為について
当Groupでは、宣伝行為は基本的には禁止しています。<br>
ただし、宣伝ガイドラインに則している場合は許可します。<br>
※[配信・投稿ガイドライン](sns_streaming.md)により、チャンネルをSepicServer Discordに記載する際もこのガイドラインに則すこととなります。<br>

!!! Warning "注意!!"
    違反した場合は、==処罰==　となります。

<br><br>

## 宣伝ガイドライン
1. {++@宣伝者++}ロールが必要です。[^1]
2. {++#宣伝++}チャンネル以外で宣伝しないでください。[^2]
3. {++機密情報を含む++}投稿はしないでください。
4. **ハラスメントや糾弾、セクシズム（性差別）、レイシズム（人種差別）、ヘイトスピーチ、誹謗中傷**等の内容を含む投稿は一切認められません。
5. URLのみの投稿は禁止。フォーマットは後述
6. URLはリンク切れをしない、無期限かつ使用回数無制限のものとする。
7. ガイドラインに則さない投稿は、修正依頼をします。<br>修正依頼後も修正されないものは削除します。[^3]
8. 悪質な投稿は、投稿を削除したのち{++@宣伝者++}ロールを剥奪します。[^4]
9. 投稿後、１週間以内に ==@sou== もしくは ==@kaka_2021== からの:white_check_mark:の印がついていることを確認してください。[^5]


[^1]: ==@sou== もしくは ==@kaka_2021== に宣伝したい旨を伝えてください。<br>審査後、{++@宣伝者++}ロールを付与しご連絡します。<br>宣伝したい旨を伝えた段階で、このガイドラインに同意したこととします。
[^2]: {++#宣伝++}チャンネル以外で宣伝した場合は削除します。<br>{++@宣伝者++}ロールを持っている方はロールを剥奪します。
[^3]: 修正依頼後、１週間以内に修正してください。
[^4]: 第４項を含む内容、連投、宣伝して即抜けなどを悪質と定義します。
[^5]: :white_check_mark:の印が1週間以上つかない場合は ==@sou== もしくは ==@kaka_2021== に連絡をしてください。
