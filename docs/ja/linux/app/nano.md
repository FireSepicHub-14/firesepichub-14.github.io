# Nano / テキストエディター

nanoエディタは**vi**よりも直感的なテキストエディターです。<br>

!!! warning "注意"
    このテキストはまだ書き途中です!!<br>

## install
```
# まず、パッケージリストを更新
sudo apt update

# nanoのインストール
sudo apt install nano -y
```


## nanoの使い方
```
user@machine:~/Documents$  mkdir sample1.txt
```

ではsample1.txtを開いてみましょう。<br>

```
user@machine:~/Documents$  nano sample1.txt
```
